# Промпт для нейронки: ПОВТОРНАЯ независимая оценка проекта (вставь целиком)

Ты — опытный продюсер/аналитик F2P мобильных игр (idle/гача), оцениваешь проекты для инвестиций и софт-лонча. Дай ЧЕСТНУЮ независимую оценку игры по 10-балльной шкале с разбивкой по аспектам, сравнение с рынком, и приоритеты. Был предыдущий аудит этого проекта с оценкой **4.7/10** (главные дыры: сломанная бесконечная прогрессия, монетизация без экономики, нет замеренных KPI, web-only). С тех пор многое сделано — оцени ТЕКУЩЕЕ состояние и скажи, на сколько проект вырос и что осталось.

## ЧТО ЭТО ЗА ИГРА
Киберпанк auto/idle-RPG на Godot 4.7 (GDScript), мобилка, F2P+гача. Соло-разработчик (россиянин, но есть тайский счёт → монетизация на Google Play открыта). Кросс-платформа: web (GitHub Pages, играбельно) + Android-нативка (AAB собран), iOS планируется через облачный CI.

### Геймплей-ядро
Отряд из 4 классов (Снайпер/Штурм/Танк/Хакер) АВТО-бьётся с волнами врагов + боссы. Лут (оружие/модули) с боссов, экипировка, ульты (тап). Прогрессия 2-слойная: уровни бойцов (золото) + множительные аугменты (ядра за престиж) + сингулярность (2-й слой). 

### ЧТО СДЕЛАНО НЕДАВНО (с момента оценки 4.7/10)
1. **БЕСКОНЕЧНАЯ ПРОГРЕССИЯ ПОЧИНЕНА** (был мёртвый потолок ~стадия 66). 3 структурных фикса: exp-хвост к perma-множителю (супер-полином бьёт экспоненту HP-стены), урон/HP int64→float64 (потолок 9.2e18→1e300 ≈ стадия 2344), perma-множитель и к HP (баланс выживания). НЕЗАВИСИМО валидировано (мат-симуляция + код-аудит + длинный бот-тест): реально бесконечно, стена остаётся жёсткой («лезешь но потеешь»), без взрыва. До максимума ~500к престижей = практически недостижимо = эффективно ∞.
2. **CASUAL-CORE УПРОЩЕНИЕ** (по фидбэку живого тестера): диагноз был «глубокий киберпанк vs казуальная idle» → выбран курс «cyberpunk vibe, casual core». Сюжет спрятан из меню (не навязывается), карта убрана, лом-реролл→простой апгрейд, ядра убраны из ачивок, UI-чистка (AUTO-кнопка, забрать-всё, батлпас серым, описания квестов).
3. **НАТИВКА ANDROID**: toolchain на сервере, debug APK + подписанный release AAB, Firebase-авторизация переведена на чистый REST (кросс-платформа), JS-bridge фиксы (ввод ника нативным диалогом).
4. **АНАЛИТИКА-АРХИТЕКТУРА**: плагг-абль слой + 7 KPI-событий (session/tutorial/stage/prestige/gacha/iap/ad), готов к ByteBrew.
5. **ОНБОРДИНГ**: лёгкий гол-баннер новичку (цель по прогрессу до 1-го престижа) + пульс ключевой кнопки.
6. **Мультиплеер-кланы** (Firebase): профили #ID, фракция-клан-боссы (общий HP, недельная ротация), лидерборды, клан-магаз, чат. **Локализация** RU/EN (двуязычная).

### ЧТО ЕЩЁ НЕ СДЕЛАНО (известные пробелы)
- Монетизация — каркас есть (алмазы/гача-pity/реклама/магазин), но нет полноценной экономики (стоки/value-ladder/LiveOps — есть блюпринт, не внедрён).
- Нет ЗАМЕРЕННЫХ KPI (аналитика внедрена, но без живых игроков данных нет).
- Анти-чит/безопасность: НЕТ (клиент-авторитет, Firebase RTDB открыт) — критично перед монетизацией.
- Контента мало (4 локации, ограниченный ростер врагов/боссов), арт неполный (часть героев/врагов — статика).
- Код — монолит (main.gd ~5500 строк).
- Звук/музыка почти нет.
