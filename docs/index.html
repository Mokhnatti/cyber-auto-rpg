00 О чём игра

Киберпанк auto/idle-RPG про отряд киборгов-бойцов. Собираешь команду, наращиваешь её кибернетическим железом, бой идёт сам — а ты оптимизируешь сборку и в трудных боях вмешиваешься руками.
Одной строкой: не «качаешь циферки» — наращиваешь киборга железом, класс за классом, имплант за имплантом.

Сеттинг — киберпанк (сознательный уход от фэнтези-болота, 90% жанра). Любимая эстетика, и здесь она не обои, а корень механики.

01 Фишка — прокачка через ИМПЛАНТЫ

Главное отличие от миллиона авто-RPG: прогрессия не «+1 уровень», а вживляешь железо в слоты тела.

🧠 Нейро — мозг, взломы 👁️ Глаза — точность, крит 🦾 Руки — оружие, урон 🫀 Торс — HP, броня 🦵 Ноги — скорость, уклон
Видно на модели. Вставил бронелицо танку — он меняется. Прогрессия = косметика = читаемость билда с одного взгляда.
Глубина без доната. Импланты падают с боёв / крафт / апгрейд по редкости (обычный → легендарный). Честная гринд-морковка.
Билды и гибриды. Снайпер с боевыми руками, танк с нейро-взломами. Узнаваемые архетипы + простор для копающих.
Синергии отряда. Импланты дают сет-бонусы между бойцами → бой решает умная СБОРКА, не задрот-гринд.

03 Прокачка — 7 слоёв с разным темпом

Секрет жанра: несколько систем тикают в разном темпе → всегда в 2 минутах от апгрейда И всегда есть далёкая цель.

1
🔼 Уровень бойца — база (HP/урон), за бой. Быстрая морковка.
2
🦾 Импланты — железо в слоты тела. БИЛД. По редкости.
3
💾 Прошивка имплантов — каждый имплант сам качается. Микро-морковка внутри железа.
4
🧬 Шасси / ранг — дубликаты → апгрейд корпуса, выше редкость, новый облик. Гача-механика БЕЗ доната.
5
🌳 Дерево аугментаций класса — ветки специализации (крит/дальнобой/дебаффы…). Идентичность билда.
6
⚙️ Оружие — отдельная ось сборки.
7
🏢 Штаб / кибердек — баффы всему отряду + idle-доход офлайн. Сердце idle-петли, аккаунт-прогрессия.

04 Боёвка — авто-файтер со СКИЛЛ-КЛАПАНОМ

Бойцы дерутся сами, ты прожимаешь ульты. А на боссах включается активный слой — клик-боксы.

Клик-боксы на боссах (успей нажать — короткое окно)

  • 🎯 Урон-окно — попал в тайминг → крит/бонус-залп
  • 🛡️ Уворот/блок-окно — успел → команда уклоняется от телеги; проспал → урон
  • ⚡ Заряд ульты — нажатия копят/ускоряют ульту
  • 💻 Класс-окна — хакер «взлом» (вырубить атаку босса), снайпер «прицел» (отметить уязвимость). Импланты улучшают эти окна → связь с прокачкой

Зачем — решает болезнь жанра

Большинство авто-RPG = чистый пассив → стенку пробиваешь только донатом/задротом. Скилл-клапан даёт пройти босса руками, когда на автомате уже не идёт.
  • Анти-p2w — пробиваешь скиллом, не кошельком
  • Отличие — в авто-RPG скилл нигде не значит ничего, а тут значит
  • Вовлечение — босс = активный данс, не «смотрю бой»
🚨 Баланс-принцип: авто проходит рутину; боссы = стена (недокач + автомат = не пройти), НО активной игрой можно компенсировать недокач. Скилл = ШАНС, не гарантия (не бог при нуле кача, но обгоняешь свою прокачку). Казуал догонит гриндом, скилловый пробьёт раньше руками — никого не выгоняем, донат не нужен.

Бонус: клик-боксы одинаково работают на тач (мобила) и ПК → не привязывает к платформе.

05 Честность — анти-pay-to-win

Принцип переносится 1:1 из прошлого проекта (Movement Parkour Racer): всё качается ЗА ИГРУ, не за донат. Стенку боссов пробиваешь скиллом (клик-боксы §04), а не кошельком. Если монетизация — то косметика / premium-покупка, НЕ лутбоксы на силу.

06 Реализм — где соло выживает

🚨 Честно: жанр ПЕРЕПОЛНЕН. Мобильный F2P-gacha = самый жестокий для соло сегмент: решает маркетинг-бюджет (UA), не геймплей. Соло без бюджета там не виден.
✅ Но соло выстреливали через premium на Steam (+порт на мобилу), нишево, на ОДНОЙ свежей механике: Melvor Idle (соло, сотни тысяч), NGU Idle, Cookie Clicker, Idle Slayer. Наша свежая механика = импланты-билдер + скилл-клапан.
Склонность: делать как premium / честную игру (Steam-first или premium-mobile), НЕ гача-доилку. Scope держать УЗКИМ — урок прошлого проекта: не раздувать, одна читаемая полка.

07 Статус и что дальше

Стадия: ранний штурм концепта (20.06.2026). Определены: фишка (импланты), 4 класса, 7 слоёв прокачки, боёвка (скилл-клапан), честность, рыночный путь, арт (2D paper-doll), движок (Godot). Кода и прототипа нет.

Тех-стек (решено)

  • Движок: Godot 4.x (GDScript; C# в кармане) — выбран под вайб-кодинг: текстовые сцены правлю напрямую, один паттерн, мгновенный цикл, бесплатно. Unity отклонён (GUID-сцены руками не править, domain reload, лицензия); Unreal отклонён (тяжёл для idle/UI).
  • Арт: 2D paper-doll — импланты наслаиваются спрайтами.
  • Инструменты Claude Code: текстовые файлы напрямую + headless-CLI + тесты GUT (MIT); MCP (Coding-Solo/GoPeak, MIT) для скринов/ошибок — опционально.
