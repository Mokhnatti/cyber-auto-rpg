# 🎲 НАША ЛУТ-СИСТЕМА — как она устроена (человеческим языком)

> Объяснение того, что РЕАЛЬНО в игре сейчас (21.06.2026). Числа — стартовые, крутятся по playtest. Теория/правила → LOOT-RULES.md, CONCEPT §14-15.

## Из чего состоит шмотка
У каждого БОЙЦА — 5 слотов имплантов (🧠 Мозг, 👁 Глаза, 🫀 Тело, 🦾 Руки, 🦵 Ноги) + 🔫 пушка.

Каждый предмет описывают ДВЕ НЕЗАВИСИМЫЕ оси:

**1. РЕДКОСТЬ = сколько у предмета стат-строк.** Задаётся в момент дропа, НЕ меняется прокачкой.
| Редкость | Стат-строк | Цвет |
|---|---|---|
| Серый | 1 | 🩶 |
| Зелёный | 2 | 💚 |
| Синий | 3 | 💙 |
| Фиолет | 4 | 💜 |

**2. ЗВЁЗДЫ (★) = прокачка ВНУТРИ редкости.** Поднимаются мерджем дублей ТОЙ ЖЕ редкости. Звёзды множат значения статов предмета.

➡️ Предмет = **модель + редкость**. «Броненосец Зелёный» и «Броненосец Синий» — это РАЗНЫЕ предметы (у каждого свои звёзды и роллы). В слоте надет ОДИН — ты выбираешь билд.

## Модели (что вообще может выпасть)
3 модели на каждый слот, у каждой свой акцент стата (можно собрать билд — напр. танку всё в HP):
| Слот | Модели (акцент) |
|---|---|
| 🧠 Мозг | Нейро-MK1 (заряд ульт), Овердрайв (заряд), Тихий-протокол (крит) |
| 👁 Глаза | Оптика-MK1 (крит), Орлиный глаз (крит), Тепловизор (урон) |
| 🫀 Тело | Реактор-MK1 (HP), Броненосец (HP), Ядро-перегрев (урон) |
| 🦾 Руки | Серво-MK1 (урон), Руки гориллы (HP), Бритвы (крит) |
| 🦵 Ноги | Привод-MK1 (скор.атаки), Тяж-опоры (HP), Гончая (скор.атаки) |

## Роллы значений (двигатель «погони»)
Когда предмет падает, КАЖДАЯ его стат-строка роллится по значению — 4 ступени по 25%:
**100% / 90% / 80% / 70% от максимума стата.**
Максимумы (★1): HP +40 · урон +8 · крит +8% · скор +8% · заряд +10%.
Пример: «Реактор Серый» может выпасть с +40 HP (топ-ролл) или +28 HP (низ). Поэтому даже найдя нужный предмет, ты гоняешь за лучшим роллом.

**Доп-строки по редкости:** Серый = только основная строка модели; Зелёный +1 случайный доп-стат; Синий +2; Фиолет +3. Больше редкость = больше осей силы.

## Дроп — что, когда, с каким шансом
- **Когда падает:** после волны. Босс (каждая 5-я волна) — гарант, +2 дубля. Обычная волна — 50% шанс дропа, +1.
- **Что падает:** 40% пушка / 60% имплант (случайный боец + случайный слот + случайная модель).
- **Редкость дропа ГЕЙТИТСЯ ПРОГРЕССОМ** (нельзя выбить топ рано):
| Редкость | Открывается с | Ориентир по времени (playtest) |
|---|---|---|
| 🩶 Серый | с порога | день 1 |
| 💚 Зелёный | волна 35 | ~2–5 ч / день 1–2 |
| 💙 Синий | волна 100 | ~15–30 ч / день 5–10 |
| 💜 Фиолет | волна 220 | ~60–120 ч / день 20–40+ |
- **Шансы между открытыми тирами:** чем выше редкость — тем реже (вес ×2 за каждую ступень вниз: при открытых до Синего примерно Серый 4 / Зелёный 2 / Синий 1).
- **Защита от невезения (bad-luck):** если 6 дропов подряд без редкого (★≥Синий) — пол редкости поднимается, цветное гарантированно капнет.
- **Перефарм:** если падает дубль той же модели+редкости с СУММАРНО лучшими роллами — он заменяет базу («ЛУЧШИЙ РОЛЛ!»), дубли/звёзды сохраняются.

## Прокачка (мердж)
- 2 дубля одного предмета (та же модель + та же редкость) + золото → +1 звезда ★.
- Стоимость звезды растёт со звёздами И с редкостью: `★-цена = уровень × 50 × редкость`. → топ-звёзды топ-редкости = очень дорого (chase на месяцы).
- Звёзды НЕ повышают редкость. Синий ★10 остаётся синим (но может бить Фиолет ★1 — есть выбор билда).

## Как это держит игрока (пейсинг)
- **Стена→гринд:** HP врагов растёт экспоненциально (×1.12/волна, боссы ×3), стоимость уровня ×1.5 — ты упираешься в стену, фармишь золото/шмот, проходишь чуть дальше. Это растягивает прогресс.
- Топ-шмот = заслуга часов/дней, не вечера (см. таблицу гейта выше).
