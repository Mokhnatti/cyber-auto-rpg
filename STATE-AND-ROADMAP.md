# Cyber Auto-RPG — состояние, пробелы и широкий план работ
*Стратегический разбор (28.06.2026). Сведено: внешняя оценка 4.7/10 + 3 ресёрча + анализ кода.*

## ЧАСТЬ 1. ГДЕ МЫ СЕЙЧАС

### ✅ Что построено и РАБОТАЕТ (сильное)
- **Боевое ядро:** 4 класса с ролями/ультами, авто-бой, волны+боссы, лут с боссов.
- **Прогрессия 2-слойная:** уровни + множительные аугменты (4 равносильных билда) + престиж (ядра) + сингулярность. *(Сейчас фикс бесконечности — фаза-1 v1.8.0.)*
- **Сюжет 8.3/10:** нарратив, мессенджер-подача квестов, карма/тон, фрагменты, 3 концовки, детектив-дело. **Дифференциатор.**
- **Кланы-мультиплеер (Firebase):** профиль #ID, клан по коду, фракция-клан-боссы (9/3 фракции, недельная ротация), общий HP + лидерборд, награды (алмазы+жетоны), клан-магаз, чат. **Сильнейший ретеншн-актив (по оценке).**
- **Монетизация-каркас:** алмазы, гача с pity, скорость-буст, реклама-бусты, магазин, реролл.
- **Локализация:** i18n, двуязычная RU⇄EN, переиспользуемый QA-сканер (под любой язык).
- **Инфраструктура:** бот-харнесс (4 тактики, 16x, телеметрия), смоук-тесты, авто-калибровка, Firebase, CI-пуши.

### ⚠️ Что СЛАБОЕ / риски
- **Прогрессия** — была сломана (потолок 66), сейчас чиним (калибруется).
- **Монетизация** — каркас без экономики (нет стоков, нет value-ladder, нет LiveOps-календаря).
- **НЕТ замеренных KPI** — ни D1/D7/D30, ни ARPU. Решения вслепую. **Корневой пробел.**
- **Web-first** — нет нативки (рефактор Firebase→REST в работе) → IAP/пуши/аналитика ограничены.
- **Контента мало** для выжигания (4 локации, ограниченный ростер).
- **Арт неполный** — снайперша анимирована (Ludo), остальные герои/враги — статика/заглушки.

## ЧАСТЬ 2. ЧТО УПУСТИЛИ (пробелы)

1. **Аналитика/телеметрия игроков** — критично, без неё не понять что чинить. (План готов: ByteBrew.)
2. **Экономика монетизации** — стоки на каждую валюту, офферы, LiveOps. (Блюпринт готов.)
3. **Онбординг/туториал** — первые 60 сек решают D1; сейчас минимальный (один help-попап).
4. **Нативная сборка** — без неё нет реального рынка. (План + рефактор Firebase в работе.)
5. **Security Rules Firebase** — сейчас test-mode (открыто до 28.07!) → перед релизом ЗАКРЫТЬ + апи-ключ ограничить.
6. **Анти-чит кланов** — сейчас «достаточно для друзей»; при росте нужен App Check + Cloud Functions (Blaze).
7. **Push-уведомления** — отложены (нужна нативка + FCM) — мощный ретеншн-крючок для idle.
8. **Анти-power-creep гачи** — политика на будущие баннеры (чтоб не обесценивать покупки).
9. **Звук/музыка** — почти нет (метал-саундтрек обсуждали, не интегрирован).
10. **Облако-сейвы/аккаунты** — анон-аккаунт не линкуется (потеря прогресса при смене устройства). Линковка Google/Apple перед сторами.
11. **Юр/приватность** — Privacy Policy, Data Safety, гача-дисклоуз, consent EU.

## ЧАСТЬ 3. ШИРОКИЙ ПЛАН РАБОТ (фазы по приоритету)

### 🔴 ФАЗА 0 — ПОЧИНИТЬ ЯДРО (сейчас, дни)
- [идёт] Бесконечная прогрессия: калибровка ботами до волнообразной кривой без плато/взрыва. *Порог: бот >500 стадий, темп престижей здоровый.*
- [идёт] Рефактор Firebase→REST (для нативки).
- Починить hoard-тактику (сломана, best=6).
- Откатить тест-алмазы (999999) перед любым релизом.

### 🟠 ФАЗА 1 — НАТИВКА + ЗАМЕР (1-3 нед)
- Android-нативка (toolchain на сервере, AAB, keystore).
- Аналитика ByteBrew + 7 ключевых событий (session/tutorial/stage/prestige/gacha/iap/ad).
- Security Rules Firebase + ограничить апи-ключ.
- Усилить онбординг (первые 60 сек).
- Закрытый internal-тест (12 тестеров/14 дней — требование Google).
- *Цель-бенчмарки: D1≥35%, D7≥12%, D30≥6%.*

### 🟡 ФАЗА 2 — ЭКОНОМИКА + LiveOps (2-4 нед)
- Стоки на каждую валюту (экспоненциальные).
- Value-ladder офферов ($0.99-$99.99) + first-purchase + сегментация.
- Батлпас в core-loop ($4.99-9.99).
- Гача-тюнинг (рейты, soft/hard pity, дисклоуз).
- Недельный LiveOps-календарь вокруг клан-боссов.
- Клан-магаз на жетоны (досдоить).
- *Цель: конверсия ≥2-3%, растущий ARPPU.*

### 🟢 ФАЗА 3 — КОНТЕНТ + АРТ + ПОЛИРОВКА (параллельно, после ядра)
- Арт/анимация остальных героев + врагов (Ludo-пайплайн).
- Больше локаций/квестов/боссов/врагов.
- Звук/музыка (метал-саундтрек).
- Облако-сейвы + линковка аккаунтов.
- Доп. языки (KO/PT-BR — сканер готов).
- Push-уведомления (FCM).

### 🔵 ФАЗА 4 — СОФТ-ЛОНЧ + ИТЕРАЦИИ
- Дешёвое гео, замер когорт, A/B по билдам.
- Если D7≥12% + положительная экономика → ФАЗА 5 (масштаб/питч издателю). Иначе — нишевый/портфолио.

### ♾ СКВОЗНЫЕ (всегда)
- QA: бот-калибровка + скрин-проверка панелей на каждый крупный кусок.
- Юр/приватность перед сторами.
- Диана — живой тест каждого билда.

## ВЫВОД
Игра — крепкий ранний прототип с РЕАЛЬНЫМИ дифференциаторами (нарратив, кланы-соц-слой). Главное сейчас: **починить ядро прогрессии** (идёт) → **нативка+замер KPI** (вслепую больше нельзя) → **экономика**. Контент/арт — НЕ раньше, чтоб не полировать фасад на чинящемся фундаменте. Все 3 этапа уже исследованы — дальше чистое выполнение по фазам.
