# CYBER AUTO-RPG — концепт (рабочее название)

> Новый проект, начат 20.06.2026. Киберпанк auto/idle-RPG.
> Рабочее название — ВЫБРАТЬ (варианты: CYBER SQUAD / AUGMENT / IRONFLESH / Корпорация / …).
> Отдельно от Movement Parkour Racer (тот проект сохранён, на паузе/осмыслении).
> ⚠️ Концепт сырой, ранний штурм. Источник правды по мере роста — этот файл.

## 0. ЧТО ЭТО
Киберпанк auto/idle-RPG про отряд киборгов-бойцов. Собираешь команду, наращиваешь её кибернетическим железом (импланты), бой идёт авто/полу-авто, ты оптимизируешь сборку. Сеттинг — киберпанк (НЕ фэнтези: сознательный уход от 90% жанра-болота, и любимая эстетика Рамиля).

## 1. РЫНОЧНЫЙ КОНТЕКСТ (честно, 20.06.2026)
- 🚨 Жанр auto/idle-RPG ПЕРЕПОЛНЕН. Мобильный F2P-gacha-сегмент = самый жестокий для соло: решает маркетинг-бюджет (user acquisition), не геймплей. Соло без UA-бюджета там не виден.
- ✅ НО соло реально выстреливали в idle-RPG через **premium на Steam (+ порт на мобилу), нишево, на ОДНОЙ свежей механике**: Melvor Idle (соло, сотни тысяч), NGU Idle, Cookie Clicker, Idle Slayer.
- **Вывод-склонность:** делать как premium / честную игру (Steam-first или premium-mobile), НЕ как гача-доилку. Фишка обязательна — иначе «ещё одна авто-RPG».
- **ПЛАТФОРМА — ОТКРЫТО:** mobile F2P vs premium Steam(+mobile). Склонность — premium (там у соло шанс).

## 2. ФИШКА (ядро отличия) — ПРОКАЧКА ЧЕРЕЗ КИБЕР-ИМПЛАНТЫ
Прогрессия не «+1 уровень», а **вживляешь железо в слоты тела**. Киберпанк — КОРЕНЬ механики, а не обои.
- Видно на модели (вставил бронелицо танку — он меняется) → прогрессия = косметика = читаемость билда с одного взгляда.
- Глубина билдов без доната. Импланты падают с боёв / крафт / апгрейд по редкости.
- Перенос идеи из Movement Parkour Racer (там протагонист = модульное кибер-тело, сменный корпус) — в паркуре было сложно, тут в самый раз.
- **Крючок №2:** импланты дают сет-синергии между бойцами → авто-бой выигрывает умная СБОРКА отряда+железа, не задрот-гринд (анти-p2w).

**МОДЕЛЬ ПРОКАЧКИ — ДВА СЛОЯ (решено 21.06.2026, идея Рамиля, тест жены):** элегантный гибрид простоты и глубины.
- **БАЗА статов = класс + надетые импланты/шмотки.** Импланты ФОРМИРУЮТ базовые статы героя.
- **УРОВЕНЬ = МНОЖИТЕЛЬ этой базы** (× ко всем статам от имплантов). Итог: `стат = база(класс+импланты) × множитель(уровень)`.
- **УРОВЕНЬ — частый/простой слой:** обычное прохождение = просто кликаешь «↑ Уровень». Понятно всем (жена-тест: казуал не вникает). Основной денежный сток.
- **ИМПЛАНТЫ — редкий/глубокий слой:** падают с боёв, **складываются/мерджатся по ЗВЁЗДАМ** (крафт). Возишься РЕЖЕ, когда захочешь — «вижу персонажа, напали импланты, скрестил, поставил посильнее». Глубина для вовлечённых, не обязаловка.
- **Почему круто:** новичок жмёт уровень и прогрессирует; копающий собирает имплант-билд (база растёт) и уровень множит сильнее. Простой экран + глубина в одном. Сохраняет понятность (жена) И киберпанк-фишку имплантов-скелета.
- **СТАТУС в демке:** упрощённо — уровень + «пушка» пер-герой (пушка = зерно базы-оружия). Полная имплант-база с мерджем-звёздами — глубокий слой на потом.

## 3. КЛАССЫ (4, классическая ролевая четвёрка + сигнатурный слот)
- 🎯 **Снайпер** — дальний бурст. Сигнатура: ГЛАЗА (оптика, крит, дальнобой).
- 🔫 **Штурмовик** — ближний/средний дамаг. Сигнатура: РУКИ (оружейные модули, шквал).
- 🦾 **Танк** — кибер-гигант, держит удар. Сигнатура: ТОРС + БРОНЕЛИЦО (реактор, пластины).
- 💻 **Хакер/Нетраннер** — контроль/саппорт. Сигнатура: НЕЙРОИМПЛАНТ/голова (дроны, взломы, дебаффы).
4 контрастных силуэта (тощий снайпер / боец / гора-танк / худой хакер с дронами) = читаемо и стильно. Кросс-билды разрешены (снайпер с боевыми руками = гибрид).

## 4. СЛОИ ПРОКАЧКИ (глубина = несколько систем с РАЗНЫМ темпом — секрет жанра)
Принцип: всегда в 2 минутах от апгрейда И всегда есть далёкая цель.
1. **🔼 Уровень бойца** — база (HP/урон), за бой. Быстрая морковка.
2. **🦾 Импланты** — железо в слоты тела (глаза/руки/торс/нейро/ноги). БИЛД. По редкости (обычный→легендарный).
3. **💾 Прошивка имплантов (софт)** — каждый имплант сам качается. Микро-морковка ВНУТРИ железа.
4. **🧬 Шасси / ранг (эволюция)** — дубликаты бойца → апгрейд корпуса, выше редкость, новый облик. Медленный престиж. Гача-механика БЕЗ доната.
5. **🌳 Дерево аугментаций класса** — ветки специализации (снайпер: крит/дальнобой/дебаффы; танк: броня/контратака/провокация). Идентичность билда.
6. **⚙️ Оружие** — отдельная ось сборки (пушки/клинки/дрон-пушки).
7. **🏢 Мета: ШТАБ / кибердек** — баффы всему отряду + idle-доход офлайн (сердце idle-петли). Аккаунт-прогрессия, самая долгая.

## 4А. МОДЕЛЬ ИГРЫ — МАКСИМАЛЬНО ПАССИВНЫЙ IDLE (решено 21.06.2026)
🎯 **Цель Рамиля: максимально пассивно для игрока.** → модель = idle-DPS-прогрессия (AFK Arena × Tap Titans 2, но БЕЗ обязательного тапа-кликера — тап = активность, против пассива).
- **Отряд бьёт сам и НЕ гибнет** (убрана смерть/баланс выживания). Fail = только «не хватило DPS на босса за таймер» → фармишь дальше.
- **💰 Золото капает само:** за киллы + пассив/сек + **ОФЛАЙН** (закрыл телефон → вернулся → забрал накопленное). Ядро пассива.
- **Прогресс = числа:** зашёл на 2 мин → вкачал золото в урон/импланты → сильнее → дальше волны → больше золота. Игрок = менеджер прокачки, игра играет сама.
- **👑 Босс = DPS-таймер-гейт (тайм-чек как TT2):** авто не вынес за время → качаешься (даже офлайн) → прошёл. Без обязательного скилла.
- 🔧 **СКИЛЛ-КЛАПАН (§4Б) → ОПЦИОНАЛЬНЫЙ бонус:** ручные ульты / опц. тап / клик-боксы на боссах ускоряют, но НЕ нужны для прогресса. Пассивный игрок проходит без касаний; активный — быстрее. Стандарт топ-idle (авто + опц. ручной буст).
- **Гача-импланты (§5) = множители DPS** — ложатся идеально.
- **РОЛИ/ПАТТЕРНЫ КЛАССОВ (решено 21.06.2026) — не «разный урон», а разный СПОСОБ атаки:**
  - 🦾 **ТАНК** — большое HP + прирост HP, **его бьют первым** (фронт-линия ромба). Низкий урон, держит.
  - 🔫 **ШТУРМОВИК** — частый урон в ОДНУ цель (single-target DPS).
  - 💻 **ХАКЕР** — «взлом»: бьёт ВСЕХ врагов по чуть-чуть (AoE, урон по площади).
  - 🎯 **СНАЙПЕР** — редко, но БОЛЬНО: долгое **время прицеливания** → большой урон. Если цель умерла за прицел → перенацеливается на нового. Burst single-target.
  - **🌀 ПАССИВНЫЕ АУРЫ КЛАССА (решено 21.06.2026) — каждый боец пока ЖИВ бафает ВЕСЬ отряд (синергия):**
    - 🦾 Танк → всем +HP (маленький %).
    - 💻 Хакер → всем быстрее заряд ульты.
    - 🔫 Штурмовик → всем +скорость атаки.
    - 🎯 Снайпер → всем +урон.
    - → теряешь бойца = теряешь и его ауру (цена смерти, стимул беречь).
  - **СТАТЫ ИГРЫ (решено 21.06.2026) — финальный набор:** ❤️ ХП · ⚔️ доп урон атаки · ⚡ скорость атаки · 🎯 шанс крита · 💥 урон крита (множитель) · 🌀 скорость набора ульты. Профиль класса задаёт базу (снайпер — высокий крит 30%/×2.2, танк — HP/мало крит). Профиль класса задаёт базу.
  - **🚨 ТАБЛИЦА СКЕЙЛИНГА (решено 21.06.2026, критично для баланса):** УРОВЕНЬ множит ТОЛЬКО плоские статы; % статы — НЕ от уровня (иначе экспон. рост сломает игру), только от шмоток/имплантов.
    - ✅ от уровня (×множитель): **ХП, урон** = (база+шмотки) × уровень.
    - ❌ НЕ от уровня (только шмотки/импланты, +ауры/ульты): **скорость атаки, шанс крита, урон крита, заряд ульты**.
    - В демке УЖЕ так: hpg/dmgg растут с уровнем, crit/critx/atk_spd — статичны к уровню. ✅
  - **ИНВЕНТАРЬ ШМОТОК — следующий слой:** даёт БАЗУ всех статов (особенно крит/скорость, которых нет от уровня). Импланты-скелет (§фишка) = эти шмотки, мердж по звёздам.
  - **УЛЬТЫ по классу (обновлено 21.06.2026):**
    - 🦾 Танк — ЩИТ всей команде.
    - 🔫 Штурмовик — всем +скорость атаки на время.
    - 💻 Хакер — мощная плюха (AoE-урон по всем врагам).
    - 🎯 Снайпер — мощный выстрел в НАЗНАЧЕННУЮ цель (тыкаешь врага). Таргет-момент (активный) — реализуется отдельно.
  - Мощь ульт растит имплант «Ядро» (§инвентарь).
- **🩸 HP-БАЛАНС (ключевой принцип, решено 21.06.2026):** уровень даёт ВЫЖИВАНИЕ (HP+статы), пушка — только урон. → «стеклянная пушка» (качаешь лишь урон) УМИРАЕТ от AoE врагов / способностей, не успев вынести пачку. Создаёт смысл качать И уровень (HP), И урон. Без HP-системы не было бы выбора. DPS-чек (успеть убить мобу до того как они убьют бойцов) — ядро баланса. Реализовано: уровень → +HP, пушка → урон.
- **ТЕРМИНОЛОГИЯ (решено 21.06.2026):** наши = **БОЙЦЫ** (PC); враги = **МОБА** (не «NPC» — тот для нейтралов). Враги масштабируются по волнам (HP/урон ×волну) → нужен рост бойцов.
- **ПРОКАЧКА ЧЕРЕЗ АП ШМОТОК (решено 21.06.2026):** вкладываешь золото в шмотку/имплант → растёт стат. Это и есть денежный сток (вместо «голой кнопки урона»). Шмотки = импланты §2 (апаются за золото). Гача даёт сами шмотки (§5), золото — апает их уровень.
- **🍀 УДАЧА (идея-фишка):** стат/перк на крит-шанс / лучший лут — в банк перков (§11 второй эшелон). Доп. ось прокачки сверх статов.

## 4Б. БОЁВКА — авто-файтер со СКИЛЛ-КЛАПАНОМ (опц. слой поверх пассива §4А; решено 20.06.2026)
Классический авто-файтер: бойцы идут и дерутся сами. Игрок прожимает **ульты** (активные способности). НО на боссах добавляется активный слой — **клик-боксы**.
**Клик-боксы (на боссах, короткие окна — успей нажать):**
- 🎯 **Урон-окно** — попал в тайминг → крит/бонус-залп.
- 🛡️ **Уворот/блок-окно** — успел → команда уклоняется от телеги босса; проспал → урон.
- ⚡ **Заряд ульты** — нажатия копят/ускоряют ульту.
- 💻 **Класс-окна** — хакер: «взлом» (вырубить атаку босса); снайпер: «прицел» (отметить уязвимое место). Импланты УЛУЧШАЮТ эти окна → прямая связь боёвки с прокачкой §4.
**ЗАЧЕМ (ядро, решает болезнь жанра):** большинство авто-RPG = чистый пассив → стенку пробиваешь только донатом/задротом. Скилл-клапан даёт пройти босса РУКАМИ, когда на автомате уже не идёт. → анти-p2w (честность как в паркуре: пробиваешь скиллом, не кошельком) + отличие от дженерика (в авто-RPG скилл нигде не значит ничего) + вовлечение (босс = активный данс, не «смотрю бой»).
**🚨 БАЛАНС-ПРИНЦИП:** авто проходит рутину/фарм; боссы = стена (недокач + автомат = не пройти), НО активной игрой можно КОМПЕНСИРОВАТЬ недокач. Скилл = ШАНС, не гарантия (не бог при нуле кача, но обгоняешь свою прокачку). Казуал догонит гриндом, скилловый пробьёт раньше руками — никого не выгоняем, донат не нужен. Соотношение «сколько скилл компенсирует» — крутить.
**Платформа-бонус:** клик-боксы одинаково работают на тач (мобила) и ПК → не привязывает к платформе.
**УЛЬТЫ — РЕШЕНО (21.06.2026): 4 кнопки-портрета внизу, канон жанра 1-в-1 (AFK Arena/Idle Heroes).** Портрет заряжается (заливка) → готов (светится) → тап = ульта. Уже в прототипе. НЕ усложняем: rage-придержание/dual-ult из WoR (§11.2) — опц. надстройка на потом, не базовая. Заряд = откат по таймеру (проще; «энергия от боя» как в AFK — опц. позже, косметика).
🚨 **ПРИНЦИП «НЕ ТЕАТР» (ресёрч 20.06):** в Legend of Mushroom runner-бой = ТЕАТР над gacha/слот-машиной (движение косметическое, реально stat-чек). Урок: чтобы не скатиться в idle-иллюзию, нужен genuine skill-слой. У нас это ИМЕННО скилл-клапан (ульты + клик-боксы) — он делает исход зависимым от рук, а не только от стат-чека. Это наше отличие от LoM-театра, держать его настоящим.

**UI/КОМПОНОВКА БОЕВОГО ЭКРАНА (ресёрч 20.06, канон auto-battler + thumb-zone UX):** мобильный портрет, 3 зоны: ВЕРХ 10-15% (статус: волна/этап, полоса босса, валюты — НЕ тапается, «красная» зона) · ЦЕНТР 55-70% (бой: отряд слева / враги справа / параллакс, HP над головой раненых — не загромождать) · НИЗ 20-30% (управление: портреты-кнопки ульт + HP + заряд, кнопка скорости x1/x2/x3 — «зелёная» зона под палец). Тап-таргеты ≥48px, отступ ≥8px. Схема «отряд слева → враги справа → сходятся в центре» = классика (AFK Arena/Idle Heroes/Legend of Mushroom) + escalation-множители по волнам (HP/урон ×волну) = бесконечный контент дёшево.

## 5. МОНЕТИЗАЦИЯ — F2P + КЛАССИЧЕСКАЯ ГАЧА (РЕШЕНО 20.06.2026, разворот)
🔄 **ПИВОТ от premium/анти-p2w → F2P + гача на силу.** Рамиль выбрал осознанно (после разбора рисков). Классический донат: **персонажи, оружие (пушки), импланты — через лутбоксы/гачу.** Родная модель жанра (auto-RPG/idle живут на гаче), потолок денег выше ЕСЛИ взлетит, платящим есть за что платить.
- **Платформа → МОБАЙЛ F2P основной** (Google Play / App Store) — там живёт гача. Steam — опц. позже. (Меняет §1 «склонность premium».)
- 🚨 **ЧТО ЭТО МЕНЯЕТ (принято осознанно, держать в уме):**
  - **Соло без UA-бюджета почти не виден** (ресёрч продаж: F2P-gacha = маркетинг-driven сегмент). → нужна вирусность / издатель / маркетинг, иначе утонет. Главный риск выбора.
  - **Скилл-клапан меняет роль:** был «анти-p2w честность» → теперь = ВОВЛЕЧЕНИЕ/удобство (пройти босса руками ИЛИ задонатить персонажа). §4Б переосмыслить (не «не театр», а активный слой).
  - **Юр: раскрытие шансов лутбоксов обязательно** (Apple/Google/Steam), лутбоксы запрещены в Бельгии/Нидерландах (гео-ограничения), РФ ужесточает. Заложить odds-disclosure.
  - **Аудитория** смещается с premium-игроков на F2P-массу + платящих китов.
- **Структура гачи (стандарт):** баннеры героев/оружия, pity-счётчик (гарант после N круток), премиум-валюта за реал + игровая за гринд, дубликаты→ранг (§4 шасси), ивент-баннеры. Косметика — отдельно.

- **Почему НЕ Unreal:** тяжёл/неудобен для idle/UI/мобилки (бинарные ассеты, жирный мобильный билд).
- 🔑 **Митигейшн единственного минуса Godot:** упрёмся в качество GDScript-генерации → пишем критичную логику на **C# внутри Godot** (.NET), сохраняя текстовые сцены и лёгкий цикл. «Лучшее из двух» без смены движка.

## 10. AI-АРТ — можно, но гибридно (ресёрч 20.06.2026)
Вопрос: строить ли арт игры на нейрогенерации. Вердикт — **ДА для статики, но НЕ сырой вывод и НЕ скрывая на Steam.**
- **🟢 Steam пускает AI** (с янв 2024), но **раскрытие ОБЯЗАТЕЛЬНО** (AI Content Disclosure, Pre-Generated, категория art → видно на стор-пейдж). H1 2025 — ~8000 раскрытий, это НОРМА.
  - 🚫 **НЕ скрывать:** нарушение Steamworks → риск снятия игры/бана аккаунта; сообщество детектит AI-артефакты и гнобит негативными обзорами за «скрытый AI». Репутационный удар > честной галочки. Скрытие к тому же не чинит копирайт-дыру (ниже).
- **🟢 Мобайл (Google Play/App Store)** — раскрытие НЕ требуется (если нет рантайм-генерации). Грузишь как есть.
- **⚠️ КОПИРАЙТ:** чисто-AI-вывод НЕ защищён (US Copyright Office янв 2025, Thaler — Верх.суд отказал март 2026; ЕС то же). Сырой AI-арт ≈ public domain → конкурент копирует 1-в-1, иска нет. Защищает только **человеческая доработка** поверх (хранить слои/before-after) + компоновка + код/нарратив. Ключ-арт/лого → товарный знак.
- **✅ ЧИСТЫЕ МОДЕЛИ:** **Adobe Firefly** — лучший для инди (лицензированные данные Adobe Stock, индемнификация на платном ~$5/мес, генерить внутри Firefly/CC). Google Imagen (Vertex) — сильная индемнификация, но enterprise. 🔴 **Midjourney** — топ-качество, худшая юр-позиция (иски Disney/Universal/Andersen, юзеру индемнификации нет). Stable Diffusion — нет индемнификации. 🚫 НЕ генерить чужое IP (персонажи CP2077, лого) — аннулирует защиту.
- **🎨 НАШ СТИЛЬ = ГЛАДКИЙ ВЕКТОР/РИСОВАННЫЙ (НЕ пиксель!) — и это дружелюбно к AI.** Весь геморрой AI-арта в ресёрче был про ПИКСЕЛЬ (диффузия портит сетку/края). Гладкий/рисованный стиль диффузия делает НАТИВНО и чисто → наш выбор стиля как раз облегчает AI-генерацию. Нюанс: AI даёт растр-PNG, не SVG-вектор — но для 2D-игры PNG в гладком стиле = то что надо.
- **🔴 ТЕХ-ДЫРА — покадровая 2D-анимация спрайтов (наш случай!):** AI плохо держит консистентность кадров (для ЛЮБОГО стиля). Статика (фоны/иконки/портреты/концепты) — решено и легко. Анимация: PixelLab (скелетный риг) / своя style-LoRA через Scenario.gg (экспорт в Godot) + РУЧНАЯ чистка. Не push-button.
- **🎯 СТРАТЕГИЯ:** Firefly + обучить свою style-LoRA на своём арте (весь набор героев в едином стиле + усиливает копирайт) → дорабатывать руками каждый ассет → раскрыть на Steam. Безопасно: фоны/иконки/портреты/UI/концепты. Рискованно: анимация спрайтов (ручной труд). В auto-RPG бой авто → спрайт-анимации мало, так что AI-статика покрывает большинство нужд.

## 11. БАНК ФИШЕК ИЗ РЕФЕРЕНСОВ (разбор 5 игр, 20.06.2026)
Топ-8 механик, точно ложащихся на наш концепт (cyber auto-RPG + импланты + скилл-клапан + гача), по ценности:

2. **🔋 Rage-бар НЕ убывает + придержание (Watcher of Realms — ⚠️ WoR это TD-гибрид, берём ТОЛЬКО ult-механику, НЕ TD-структуру/расстановку-оборону).** Заряд ульты стоит залитым; ценность в ТАЙМИНГЕ слива (придержать 2-3 героя → залп в окно стана), не в нажатии. Один бой = два уровня мастерства без микроменеджмента. + Dual-ult: auto-ult (мелкий, держит бой idle-friendly) + manual-ult (мощный, сам). Механика жанро-независима (та же energy-логика, что в AFK Arena — наш жанр).
3. **🛡️ i-frame по телеграфу босса = бинарная ставка (Lootfiend).** Наши клик-боксы на боссах → окно по началу анимации атаки: успел → неуязвимость + контр-бёрст, опоздал → урон. Жив/мёртв, читается без обучения. Один ввод = защита + рывок + бёрст (не скучная чисто-защита).

Гача-импланты ОСТАЮТСЯ при сбросе → монетизация не сгорает. Голую силу — за rebirth-петлёй, гачу — за редкость/косметику/QoL.

Двойная валюта (free за гринд + paid за реал), никогда paid→free.
