# Промпт для VISION-нейронки: оценка UI/UX по скриншотам (вставь + приложи картинку-монтаж панелей)

Ты — ведущий UI/UX-дизайнер мобильных F2P idle/гача-игр (проектировал интерфейсы тайтлов с высоким ретеншном). К промпту приложен МОНТАЖ скриншотов всех ~23 экранов/панелей нашей игры (мобилка, портрет 600×960). Дай КОНКРЕТНУЮ экспертную оценку UI/UX с прицелом на КАЗУАЛЬНОГО игрока и удержание. Не общие слова — конкретные проблемы по экранам + как исправить.

## КОНТЕКСТ
Киберпанк idle/гача auto-RPG (Godot, мобилка, F2P). Цель — КАЗУАЛЬНЫЙ игрок, минимум трения, понятность с первого взгляда. Сейчас UI функциональный, но «программистский» (без арта, базовые панели). Главный экран: отряд из 4 бойцов авто-бьётся с волнами, внизу кнопки прокачки/экипировки/престижа/«ещё», вверху ресурсы + кнопки скорости/авто/квестов/батлпаса.

## ЧТО ОЦЕНИТЬ (по 10-балльной + конкретика по экранам)
1. **ГЛАВНЫЙ ЭКРАН (ru_00_main):** читается ли что происходит? Кнопки/ресурсы понятны? Что перегружает/мешает? Иерархия внимания (где глаз)?
2. **РАСПОЛОЖЕНИЕ КНОПОК:** удобно ли для большого пальца (thumb-zone мобилы)? Что слишком мелко/близко/далеко? Важные действия доступны?
3. **ПОНЯТНОСТЬ ПАНЕЛЕЙ:** гача/магазин/престиж/экипировка/квесты/батлпас — с первого взгляда ясно что делать? Где путается казуал?
4. **КОНСИСТЕНТНОСТЬ:** единый ли стиль панелей (отступы/шрифты/цвета/кнопки)? Что выбивается?
5. **ИНФОРМАЦИОННАЯ ПЛОТНОСТЬ:** где перегруз текстом/цифрами? Где не хватает подписей/контекста?
6. **МОБИЛЬНАЯ ПРИГОДНОСТЬ:** размеры тапов (мин 44px), читаемость на телефоне, не обрезается ли.
7. **ОНБОРДИНГ/ПЕРВОЕ ВПЕЧАТЛЕНИЕ:** поймёт ли новичок за 30 сек что делать?
8. **МОНЕТИЗАЦИЯ-UI:** заметны/привлекательны ли точки трат (гача/магазин/премиум)? Не агрессивно ли / не спрятано ли?

## ВЫХОД
По каждому пункту: оценка + КОНКРЕТНЫЕ проблемы (назови экран по имени файла) + фикс. Плюс раздел «**ТОП-5 UI-ФИКСОВ С МАКС-ЭФФЕКТОМ**» (что поправить первым для казуала/ретеншна, минимум усилий). Будь конкретен и честен — это для соло-разработчика, нужны прикладные правки, а не теория.

(Имена файлов в монтаже: ru_00_main = главный, ru_gacha, ru_shop, ru_prestige = усиления/престиж, ru_gear = экипировка, ru_upgrade = прокачка, ru_daily = дейли-квесты, ru_battlepass, ru_clan/clanboss/clanchat/clanshop, ru_settings, ru_stats, ru_singularity, ru_achievements, ru_ads, ru_speed, ru_messages/dossier/case/finale = сюжет, ru_map.)
