## 2. СОСТОЯНИЕ ИГРЫ — что РЕАЛИЗОВАНО и работает
**Бой:** отряд (ромб-формация) слева, враги волнами справа, сходятся, авто-атака. Босс каждые 5 волн (полоса HP сверху). Золото пассивно (+2/с) + за килл. Кнопка скорости x1/x2/x3.

**4 класса** (`HEROES` const в main.gd), каждый: hp/dmg/atk(скорость)/atk_type/hpg/dmgg/crit/critx/ult/ult_cd/wname/wicon:
- 🎯 СНАЙП (snipe): высокий урон/крит, ульта `burst` = ТАП-ТАРГЕТ (жмёшь ульту → «ВЫБЕРИ ЦЕЛЬ» → тапаешь врага → мощный выстрел). Оружие 🔭 Рельса-винтовка.
- 🔫 ШТУРМ (single): быстрая атака, ульта `barrage` = +скорость атаки отряду. Оружие 🔫 Штурм-ган.
- 🦾 ТАНК (tank): много HP, ульта `shield` = щит+хил команде. Оружие 💥 Тяж-орудие.
- 💻 ХАКЕР (aoe): AoE-атака, ульта `hack` = AoE-плюха всем врагам. Оружие 📡 Взлом-дрон.

**Ауры классов** (`_recalc_auras`): живой танк→+10% HP отряду, снайп→+8% урон, штурм→+10% скор.атаки, хакер→-18% кд ульт.

**ДВУХСЛОЙНЫЙ ПРОГРЕСС (ключевая механика):** `итоговый стат = база(класс+оружие+импланты) × множитель(уровень)`.
- 📊 **Уровень** (`_upgrade_level`, за золото) — множитель HP/урона. Кнопка ПРОКАЧКА (список 4 бойцов). Растёт часто, просто.
- 🔫 **Оружие** (пер-класс, `_merge_weapon`) — ГЛАВНЫЙ урон. База. Качается МЕРДЖЕМ: 2 дубля + золото → +звезда.
- 🦾 **Импланты** (ПЕР-ПЕРСОНАЖ, `_merge_impl`, `IMPL_DEFS`: core/arms/optic/legs/neuro) — база статов. У КАЖДОГО бойца свои 5 слотов. Качаются МЕРДЖЕМ: 2 дубля + золото → +звезда. core→+HP, arms→+урон, optic→+крит, legs→+скор.атаки, neuro→слот есть (заряд ульт, лёгкая привязка).
- Скейлинг (`_recalc_hero`): уровень множит ТОЛЬКО HP/урон; крит/скорость — от имплантов (база, не множатся).

**Дроп** (`_drop_implant`, после волны; босс гарант 2 дубля, волна шанс 1): 40% оружие / 60% имплант — ПОД КОНКРЕТНОГО случайного бойца+слот. Копишь дубли → мердж.

**Экраны/кнопки:**
- 📊 ПРОКАЧКА — список 4 бойцов, кнопка уровня каждому (чистые уровни, оружие отсюда УБРАНО).
- 🦾 ЭКИПИРОВКА — выбор бойца портретом сверху (подсветка активного) → его ОРУЖИЕ (золотая строка) + 5 слотов имплантов. Всё пер-персонаж.
- Закрытие панелей: кнопка ЗАКРЫТЬ (выше Рестарта) + тап по фону.
- Рестарт, 4 кнопки ульт снизу.

**Технич:** фикс-шрифт DejaVuSans + NotoColorEmoji (цветные эмодзи в web). `smoke_test.py` гоняется после каждой правки — последний прогон 0 ошибок.

## 3. ДИЗАЙН-РЕШЕНИЯ этой сессии (всё в CONCEPT.md подробно)
1. Пивот: Movement Parkour Racer (UE5) → **cyber-auto-rpg** (Godot, мобилка, гача).
2. Гладкий вектор (не пиксель). Крупные NPC, HP над головой только у раненых, босс полоской сверху.
3. Прогресс: 2 слоя (уровень=множитель × шмотки=база). Таблица скейлинга: уровень множит HP/урон; крит/скор — от шмоток.
4. Импланты = **МЕРДЖ** (2 одинаковых → +звезда + золото), НЕ прямой золото-ап. Дубли с боссов/волн.
5. Импланты — **ПЕР-ПЕРСОНАЖ** (одеваешь на каждого, свои 5 слотов). Оружие — пер-класс. Уровень — пер-герой.
6. Снайпер-ульта = тап-таргет по врагу.
7. Ульта = свойство ПЕРСОНАЖА (при коллекции героев каждый принесёт свою). Уже data-driven.
8. **«Разные персонажи»** (план): отряд не фикс 4 класса — коллекция/смена героев (гача-набор). Экран ЭКИПИРОВКИ уже под это (выбор героя).
9. Стиль киберпанка СВОБОДЕН юридически (жанр не копирайтится); нельзя конкретные IP (персонажи/лого/назв. франшиз).
10. AI-арт: Firefly + своя style-LoRA, дорабатывать руками (CONCEPT §10).

## 7. GDScript грабли (чтоб не наступать снова)
- `:=` НЕ выводит тип из Dictionary/Variant-доступа → явный тип: `var x: int = dict["k"]`.
- Функция, возможно возвращающая null → аннотировать `-> Variant`.
- Эмодзи в web: нужен NotoColorEmoji.ttf как fallback в теме (уже сделано).
- Клик по врагу: `get_global_mouse_position()` (НЕ viewport) — иначе со stretch расходится.
