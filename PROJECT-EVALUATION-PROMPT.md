## ПРОЕКТ: «Cyber Auto-RPG» (рабочее название)

**Жанр:** киберпанк авто/idle-RPG (auto-battler + idle/incremental + гача). Мобилка, F2P.
**Движок:** Godot 4.7 (GDScript), веб-экспорт (HTML5) + план нативки Android/iOS.
**Платформа сейчас:** прототип на GitHub Pages, итерации по скринам.
**Команда:** соло-разработчик + AI-агент. Срок: ~несколько дней на текущий объём.

### Геймплей-ядро
- Отряд из 4 классов (Снайпер/Штурмовик/Танк/Хакер), каждый со своей ролью (одиночный урон/крит, стабильный ДПС, танк, AoE) и УЛЬТОЙ (у снайпера — тап-таргет мега-крит).
- АВТО-бой с волнами врагов: 4 норм-волны + босс-ворота на стадию. Шмот падает только с боссов.
- Бесконечные стадии вглубь, растущая сложность.

### Прогрессия (2-слойная)
- Уровень бойца = множитель; оружие+модули = база; мердж дублей.
- Усиления урон/HP/крит/скорость — МНОЖИТЕЛЬНЫЕ (4 равносильных ДПС-билда: урон/крит/скорость/танк).
- Броня = множительный DR (живучесть EHP), стекляшка-билды.
- ПРЕСТИЖ: сброс ради ЯДЕР → перма-аугменты (детерминир. выбор). 2-Й СЛОЙ престижа — «Сингулярность» (с поздних стадий).
- ⚠️ ИЗВЕСТНАЯ ПРОБЛЕМА (на оценку): сейчас мягкий потолок ~стадия 66 — престиж/сингулярность асимптотят ниже экспоненты HP врагов. Цель — переделать на бесконечную прогрессию (компаундные вложенные слои).

### Сюжет/контент
- Нарратив прошёл внутренний ресёрч-цикл, оценён двумя независимыми AI на 8.3/10. Киберпанк-история: курьер Вектор, фиксеры, twist с имплантом PHANTOM-LIMB, детективное «дело», 9 фрагментов памяти (3 фейковых), система кармы/тона (эмпатия/злость/холод), 3 концовки.
- Доставка квестов — через МЕССЕНДЖЕР-чат (облачка, фиксеры шлют задания) — необычная подача.
- 4 локации (трущобы/корп/доки/ядро) с тематическими врагами (чиби-арт + палитра CP2077).
- Дейли-квесты, достижения, батлпас, стрик-награды.

### Монетизация (F2P)
- Премиум-валюта (алмазы), гача с pity-системой, буст скорости, реклама-бусты, магазин, реролл.

### Мультиплеер: КЛАНЫ (через Firebase, реализовано)
- Анонимный аккаунт (#ID), создать/вступить в клан по 6-значному коду (до 20).
- КЛАН-БОССЫ: 9 боссов из 3 фракций, недельная ротация; общий HP-пул, каждый бьёт по своей пик-мощи (prestige-proof снапшот), realtime HP-бар + лидерборд вклада; награды по вкладу (скромные алмазы + клан-жетоны); анти-чит через Security Rules + sanity-cap (честно: на клиентской web-игре только «достаточно для друзей»).
- Клан-магаз (тратить жетоны), клан-чат (realtime).
- Кросс-платформенно (web сейчас, нативка потом — Firebase даёт push-уведомления).

### Локализация
- Система i18n (таблица строк + переключатель), полностью двуязычная RU⇄EN (переключается в настройках), проверена авто-сканером (заходит во все 22 окна, ловит утечки языка). Готова под любой язык (KO/PT-BR/...).

### QA/инфраструктура
- Бот-харнесс самоигры (4 headless-бота, разные тактики, 16x ускорение, телеметрия) для балансировки.
- Авто-смоук-тесты, скрин-проверка панелей.
