## 0. TL;DR — рекомендованный путь (для занятых)

**Базовый пайплайн (Tier A, near-free, ~$10–30 за месяц + время):**

1. **Стиль-якорь:** 1 раз сгенерить «арт-библию» — 1 эталонного героя в финальном стиле (Midjourney v7 trial / Leonardo free / Flux на ComfyUI). Зафиксировать палитру, толщину линии, свет, ракурс.
2. **Герои (4 класса):** Scenario.gg или Leonardo с *обученной на эталоне моделью* (style-LoRA) → один чистый base-спрайт на класс в одной позе. Силуэты разводим промптом (снайпер — длинный ствол/плащ; штурм — броня/каркас; танк — массивный экзоскелет; хакер — тонкий, дроны/голо-визор).
3. **Враги (8 типов + боссы):** та же обученная модель, пакетная генерация. Враги легче — меньше деталей, читаемый силуэт.
4. **Анимация:** AI НЕ делает консистентную покадровую анимацию. Берём 1 base-спрайт → риг в **DragonBones (бесплатно)** или Spine → экспорт в покадровые PNG (idle/walk/hit), как уже устроено в `prototype/sprites/heroX/`.
5. **Фоны:** Midjourney/Flux в горизонтальном формате (`--ar 16:9`/21:9), 2–3 слоя параллакса. Это место, где AI силён больше всего.
6. **UI-скин (панели/кнопки/иконки магазина, гачи, баттлпасса):** **Canva** (у владельца есть интеграция) + Recraft (нативный SVG) для иконок валют/гир. Affinity Designer (теперь бесплатен) для финальной нарезки/слайсов.
7. **Иконки гира/валют:** Recraft (SVG) или Scenario «icon»-модель; фон убрать — прозрачный PNG.
8. **App icon:** IconikAI / Recraft (стиль neon) → экспорт всех размеров iOS/Android.
9. **Чистка/прозрачность:** Photopea (бесплатный Photoshop в браузере) + bgclear.ai / Pixelcut (бесплатное удаление фона).

**Кто за что отвечает (шпаргалка):**

| Тип ассета | Главный инструмент (Tier A) | Альтернатива |
|---|---|---|
| Герои (4) | Scenario / Leonardo (обученная модель) | Flux.1-dev LoRA в ComfyUI |
| Враги (8+боссы) | то же, пакетно | Midjourney + Omni-Reference |
| Фоны (параллакс) | Midjourney v7 / Flux | Leonardo |
| Анимация | DragonBones (free) → кадры | Spine ($379) |
| UI панели/кнопки | Canva | Affinity Designer (free) |
| Иконки валют/гира | Recraft (SVG) | Scenario icon-model |
| App icon | IconikAI / Recraft | Canva |
| Прозрачность/чистка | Photopea + bgclear.ai | remove.bg / Krita |
| Готовые заглушки | Kenney (CC0), itch.io | GameDev Market |

## 1. AI-генерация изображений для игровых ассетов

### Главная проблема и её решение — КОНСИСТЕНТНОСТЬ
Один красивый спрайт умеет сгенерить кто угодно. Killer-проблема — чтобы 4 героя + 8 врагов + боссы выглядели из ОДНОЙ игры. Три рабочих рычага в 2026:

- **Trained models / Style-LoRA** (лучший рычаг): обучить модель на 10–30 картинках своего стиля → потом генерить хоть сотни ассетов «в стиле». Это делают Scenario, Leonardo, Layer.ai и локально ComfyUI (Kohya LoRA).
- **Reference images** (быстрый рычаг): Midjourney Omni-Reference (`--oref`), IP-Adapter в SD — подсунуть эталон, держать персонажа/стиль.
- **Seed + фиксированный промпт-шаблон + Style Reference (`--sref`)**: повторяемость без обучения.

### Сравнение инструментов (2026)

| Инструмент | Цена | Сильная сторона | Слабости / лицензия |
|---|---|---|---|

**Что для чего:**
- **Спрайты персонажей:** Scenario / Leonardo (обученная модель) — консистентность важнее «вау». Локально — Flux-schnell/SD + LoRA.
- **Фоны:** Midjourney v7 или Flux — тут AI объективно силён, бери лучший вайб.
- **Иконки:** Recraft (SVG) или Scenario icon-модель.
- **Текст/баннеры/лого:** Ideogram или Flux.

**Прозрачный фон / вырезание спрайта:**
- Leonardo — тогл прозрачности нативно. MJ/Flux — нет, режем после.
- Бесплатные вырезалки 2026: **bgclear.ai** (без регистрации, без вотермарка, безлимит), **Pixelcut**, EzRemove, Pokecut, Photoroom, Recraft BG-remover. Для волос/мелкой кромки — Removal.AI.

**Спрайт-листы / кадры анимации — честная правда 2026:**
AI хорош в ОДНОМ спрайте, но **плохо держит персонажа через run/jump/attack** — детали «плывут» между кадрами. Рабочий путь: сгенерить 1 чистый base-спрайт → анимировать ригом/интерполяцией (см. §2). Полу-AI-вариант: ComfyUI = base-кейфрейм + ControlNet OpenPose + character-LoRA + AnimateDiff → 16-кадровый цикл с сохранением стиля (требует возни и GPU).

**Лицензии — кратко (проверять перед релизом!):**
- Scenario, Leonardo (платно), Midjourney (платно), Flux-schnell/Pro, Recraft, Ideogram — **коммерция разрешена**.
- **Flux.1-dev — НЕ для коммерции.** SD/LoRA — зависит от конкретной модели (читать карточку на CivitAI/HF).
- В большинстве юрисдикций чисто-AI-картинка может НЕ иметь копирайта — но для F2P-мобилки это не блокер.

## 2. AI-to-Game: от картинки до Godot

Текущий формат проекта: покадровые PNG на анимацию — `prototype/sprites/heroX/idle_0..5.png`, `walk_0..7.png`, `hit_0..2.png`. Значит финал любого пути — **набор PNG-кадров**.

**Пайплайн:**
1. **Генерация** base-спрайта (Scenario/Leonardo/Flux), желательно сразу прозрачный фон или нейтральный.
2. **Чистка:** Photopea (бесплатный браузерный Photoshop, .psd) или **Krita** (free, мощная) — подчистить кромки, поправить палитру под style-guide, выровнять pivot/центр.
3. **Удаление фона** (если не было): bgclear.ai / Pixelcut → прозрачный PNG.
4. **Анимация статичного арта** — выбери ОДИН путь:
   - **DragonBones** — **бесплатный, open-source**, mesh-деформация, экспорт в спрайт-листы/кадры PNG. Лучший выбор для нулевого бюджета.
   - **Spine** — индустриальный стандарт, $379 one-time (Pro), mesh-деформация, плагин рантайма под Godot. Бери если денег чуть есть и хочешь pro-результат.
   - **Live2D Cubism** — деформирует исходную иллюстрацию целиком (сохраняет каждый мазок), идеален для гача-портретов/баннеров и «живого» героя на экране экипировки; free-версия ограничена.
   - **Cascadeur** — AI-ассист keyframe, но это 3D (free для non-commercial, Indie ~$96/год). Для 2D-спрайтов избыточно, мимо.
5. **Экспорт в кадры** из риг-тула → нарезать по той же схеме имён, что уже в проекте → положить в `prototype/sprites/`.
6. **Импорт в Godot 4:** PNG → выключить фильтрацию для пиксель-стиля (Import → Filter Off) либо оставить для гладкого; собрать `SpriteFrames`/`AnimatedSprite2D`. Частицы/«juice» — встроенный `GPUParticles2D` + аддитивный бленд для неона (генерить арт не нужно, это код+шейдер).

**«Juice» (хит-эффекты, вспышки, частицы):** делается в самом Godot (particles, screen-shake, flash-shader), плюс можно подмешать AI-сгенеренные glow/spark текстуры на прозрачном фоне. Это не «рисование», а технический эффект — отдельная мелкая задача, не блокер арта.

## 5. Найм фрилансера (Fiverr / Upwork / ArtStation)

**Ставки 2026 (Upwork/Fiverr):**
- Часовая: медиана $25/ч; мид $25–60/ч; синьор-концепт/чар-арт $60–120+/ч.
- Фикс за единицу:
  - Простой концепт персонажа: **$50–150**
  - Детальная иллюстрация: **$100–300**
  - Полный character-sheet (повороты/эмоции): **$250–800**
  - Спрайт-лист (анимация): **$100–500+**

**Прикидка для нашего набора** (4 героя + 8 врагов + боссы + UI-кит + иконки + фоны):
- Бюджетный фриланс (Fiverr-мид, спрайты без сложной анимации): **~$1 500–3 500**.
- Качественный когезивный сет с анимацией и UI-китом (синьор): **~$5 000–12 000+**.
- Умный гибрид: нанять ОДНОГО художника только на **style-guide + 4 эталонных героя** (~$600–1500), остальное (враги/фоны/иконки) добить AI «в стиле» эталона. Лучшее соотношение цена/когезия.

**Как брифовать (критично для когезии):**
1. **Style guide / mood board:** референс-доска (Pinterest/ArtStation), палитра HEX, толщина линии, уровень детализации, освещение (неон-рим-лайт), ракурс камеры (3/4 или профиль для side-scroll).
2. **Reference sheet:** размер канваса, pivot, целевое разрешение, формат (прозрачный PNG / спрайт-лист), список анимаций (idle/walk/hit) с числом кадров — как в `prototype/sprites/`.
3. **Силуэт-бриф:** для каждого класса — читаемый силуэт (снайпер/штурм/танк/хакер), цвет-код фракции.
4. Договор: коммерческие права/выкуп (work-for-hire), без роялти.

## 6. Style Guide — как залочить когезию (главное против «AI-каши»)

Инконсистентность убивает AI-арт. Лочим ДО массовой генерации:

1. **Палитра (зафиксировать HEX):** напр. фон — тёмный десатур (deep indigo/charcoal), акценты — 2 неона (cyan #00E5FF + magenta #FF1E8E), опасность — токсик-зелёный/янтарь. Каждый класс — свой ID-цвет рим-лайта.
2. **Свет:** единый закон — тёмная база + неоновый rim-light + bloom/glow. Один источник-настроение на все ассеты.
3. **Линия/детализация:** реши раз — clean-vector vs painterly vs pixel. Не смешивать.
4. **Ракурс/масштаб:** один угол камеры, одна высота персонажа в пикселях, один pivot.
5. **Reference board:** 5–10 утверждённых эталонов = «арт-библия».
6. **Технический рычаг:** обучить ОДНУ style-модель (Scenario/Leonardo/LoRA) на этих эталонах и генерить ВСЁ через неё; фиксировать промпт-шаблон + seed + `--sref`/Omni-Ref.
7. **Промпт-шаблон** (пример-каркас):
   `<class/enemy>, full body, side view, cyberpunk, neon rim light cyan and magenta, dark desaturated background, clean stylized shading, [ID-color] accent, consistent line weight, game sprite, transparent background --ar 1:1 --sref <STYLE_ID> --oref <HERO_REF>`
8. **QA-проход:** все ассеты рядом на одном экране → выбивающиеся перегенерить/подкрасить в Photopea под палитру.

## 7. Три тира пайплайна под cyber-auto-rpg

### Tier A — почти бесплатно DIY (рекоменд. старт) — ~$0–30, ~1–3 недели
- **Стиль:** обучить 1 style-model — Leonardo (free-tier хватит на старт) или Scenario trial; либо Flux-schnell/SD + LoRA локально (если есть GPU).
- **Герои/враги:** генерить через эту модель, по 1 base-спрайту, силуэты — промптом.
- **Анимация:** **DragonBones (free)** → кадры PNG → в проект.
- **Фоны:** Midjourney trial / Flux / Leonardo, 16:9 параллакс.
- **UI:** **Canva** (интеграция) + **Affinity Designer (free)** нарезка; иконки — **Recraft free** (SVG).
- **App icon:** IconikAI/Recraft free.
- **Чистка/фон:** Photopea + bgclear.ai.
- **Заглушки сразу:** Kenney CC0 для UI/частиц/иконок, пока генеришь фирменное.
- **Итого:** ~$0, максимум $10–30 если взять 1 платный месяц Leonardo/Scenario/MJ для качества. Узкое место — твоё время и возня с консистентностью.

### Tier B — умеренный (рекоменд. для качества) — ~$50–200/мес + ~$600–1500 разово, ~2–4 недели
- Платная подписка **Scenario ($20–60)** ИЛИ **Leonardo + Midjourney Standard ($30)** для топ-фонов.
- Обученная модель на эталонах = серьёзная когезия.
- **Spine ($379 one-time)** для качественной анимации (или остаёшься на DragonBones).
- **Нанять 1 художника** на style-guide + 4 эталонных героя (~$600–1500), дальше всё добиваешь AI «в стиле».
- UI-кит — Canva Pro + Affinity.
- **Итого:** ~$700–2000 разово + ~$50–200/мес пока генеришь. Лучшее цена/качество.

### Tier C — нанять художника (если есть бюджет) — ~$3000–12000+, ~1–3 мес
- Один фрилансер/мини-студия (ArtStation/Upwork-синьор) на весь когезивный сет: 4 героя с анимацией, 8 врагов + боссы, фоны, UI-кит, иконки, app icon.
- Ты даёшь style-guide и reference sheet (§5–6), контролируешь когезию.
- **Итого:** $3–5k бюджетно / $8–12k+ за полный pro-сет с анимацией. Максимум качества и оригинальности, минимум твоего времени.

## 8. Конкретный план действий (шаги)

1. **Залочить Style Guide (§6):** палитра HEX, свет, линия, ракурс, 5–10 эталонов. (0.5–1 день)
2. **Сгенерить 1 эталонного героя** в финальном виде (MJ/Leonardo/Flux), утвердить как «арт-библию». (0.5 дня)
3. **Обучить style-model** на эталоне(ах) — Scenario/Leonardo/LoRA. (~1 ч обучения + подбор)
4. **4 героя:** base-спрайты через модель, развести силуэты, прозрачный фон, чистка в Photopea. (2–3 дня)
5. **8 врагов + боссы:** пакетно через ту же модель. (2–3 дня)
6. **Анимация:** риг 4 героев + ключевых врагов в DragonBones/Spine → экспорт кадров по схеме `prototype/sprites/`. (3–5 дней — самое долгое)
7. **Фоны:** 2–3 локации, параллакс-слои. (1–2 дня)
8. **UI-скин в Canva + Affinity:** панели/кнопки магазина, гачи, баттлпасса, ачивок, дейлисов, престижа. (3–5 дней)
9. **Иконки** гира/валют (Recraft SVG) + **app icon** (IconikAI). (1–2 дня)
10. **«Juice» в Godot:** частицы/вспышки/shake — код+шейдеры, опц. AI glow-текстуры. (1–2 дня)
11. **QA-когезия:** все ассеты на одном экране, выбивающееся — перегенерить/подкрасить. Затем `smoke_test.py`. (1 день)

**Оценка Tier A:** ~$0–30, ~2–3 недели работы по вечерам. Главный риск/узкое место — консистентность героев и ручная анимация; на них уйдёт больше всего сил.
